第一问

对于第一问来说，a c组与b d组的求解过程是对称的，只是对于系数符号有所变换，因此在此只给出a c组的求解，b d组只需要在所求解的结果上更改部分符号便可

\[ \begin{equation*} \begin{aligned} e&=m_{1}a+m_{2}b \\ f&=m_{3}a+m_{4}b\\ g& = m_{1}c+m_{2}d\\ h&=m_{3}c+m_{4}d\\ b&=(e+f-(m_{1}+m_{3})a)/(m_{2}+m_{3})\\ d&=(g+h-(m_{1}+m_{3})c)/(m_{2}+m_{3})\\ z_{(1)}&=ie+jf\\ &=(im_{1}+jm_{3})a+(im_{2}+jm_{4})b\\ &=(im_{1}+jm_{3})a+(im_{2}+jm_{4})(e+f-(m_{1}+m_{3})a)/(m_{2}+m_{3})\\ &=(im_{1} + jm_{3} - ((m_{1} + m_{3})(im_{2} + jm_{4}))/(m_{2} + m_{3}))a + ((e + f)(im_{2} + jm_{4}))/(m_{2} + m_{3}) \\ ia&= i(z_{(1)}-((e + f)(im_{2} + jm_{4}))/(m_{2} + m_{3}))/(im_{1} + jm_{3} - ((m_{1} + m_{3})(im_{2} + jm_{4}))/(m_{2} + m_{3})) \\ z_{(2)} &=kb+ld=(km_{1}+km_{3})c+(lm_{2}+lm_{4})d\\ &=(km_{1}+km_{3})c+(lm_{2}+lm_{4})(g+h-(m_{1}+m_{3})c)/(m_{2}+m_{3})\\ &=(km_{1} + km_{3} - ((m_{1} + m_{3})*(lm_{2} + lm_{4}))/(m_{2} + m_{3}))c + ((g + h)(lm_{2} + lm_{4}))/(m_{2} + m_{3}) \\ jc&=j(z_{(2)}- ((g + h)(lm_{2} + lm_{4}))/(m_{2} + m_{3}))/(km_{1} + km_{3} - ((m_{1} + m_{3})*(lm_{2} + lm_{4}))/(m_{2} + m_{3})) \\ ia+jc&= i(z_{(1)}-((e + f)(im_{2} + jm_{4}))/(m_{2} + m_{3}))/(im_{1} + jm_{3} - ((m_{1} + m_{3})(im_{2} + jm_{4}))/(m_{2} + m_{3})) + j(z_{(2)}- ((g + h)(lm_{2} + lm_{4}))/(m_{2} + m_{3}))/(km_{1} + km_{3} - ((m_{1} + m_{3})*(lm_{2} + lm_{4}))/(m_{2} + m_{3})) \end{aligned} \end{equation*} \]

第二问

由于第二问和第一问的数据并无变化，因此我认为不需要对第二问直接进行求解，只需要将\(z^{新}\)与\(z^{旧}\)的变化映射写出，便可归结于第一问的问题求解

其新旧函数的变化映射可写为: \[ \begin{equation*} \begin{aligned} z^{旧}_{1}=ie+jf = z^{新}_{1}-jz^{新}_{2}/k-jg-jlh/k\\ z^{旧}_{2}=kf+lh = kz^{新}_{1}/j+z^{新}_{2}-kg-kie/j \end{aligned} \end{equation*} \] 将其带入第一问求解的a b c d中便可，具体的值就不在此赘述了
